# FORMA LIVING

A high-end, handcrafted furniture studio concept built with Next.js, blending Scandinavian minimalism with warm, tactile luxury.

## Features

- **Modern Stack:** Built with Next.js 16 (App Router), React, and TypeScript.
- **Styling:** Tailwind CSS V4 for utility-first styling with a custom design system.
- **Animations:** Framer Motion for scroll-linked animations, entrance effects, and micro-interactions.
- **Components:** Modular sections including a parallax Hero, staggered Product Grid, interactive Material Selector (pure CSS/SVG), and Showroom Locator.
- **Performance:** Optimized images with `next/image` and static page rendering.
- **Accessibility:** Semantic HTML, focus states, and aria-labels included.
- **Responsive:** Mobile-first approach scaling beautifully to 4K desktop screens.

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run the Development Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.

3. **Production Build:**
   ```bash
   npm run build
   npm run start
   ```
